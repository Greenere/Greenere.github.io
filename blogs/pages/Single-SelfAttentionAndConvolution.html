This raises the question: do learned attention layers operate similarly to convolutional layers? This work provides evidence that attention layers can perform convolution and, indeed, they often learn to do so in practice.

From a theoretical perspective, we provide a constructive proof showing that self-attention layers can express any convolutional layers.

Specifically, we show that a single multi-head self-attention layer using relative positional encoding can be re-parametrized to express any convolutional layer.

Our results seem to suggest that localized convolution is the right inductive bias for the first few layers of an image classifying network.

They theoretically prove the superiority of self-attention over convolution.

Using this experiment, they intend to state that for a fully-attentional model trained for images, the lower layer automatically simulates the behaviors of some convolutional layers.

Inspirations

They construct a relationship between multi-head self-attention layers and convolutional layers theoretically. They state that the multi-head self-attention layer has the capability to learn a equivalent convolutional layer. They also use experiments to demonstrate that it's possible to have the multi-head self-attention layer behave like convolutional layers, and in the fully-attentional model for images, the lower layers also works like convolutional layers.

In short, a convolutional layer of kernel size can be interpreted as a multi-head self-attention layer with heads and an attention map of receptive fields for each pixel.
